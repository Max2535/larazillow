Custom Workspace Settings

You can configure VSCode PER PROJECT.

Inside the project create the .vscode folder. Inside the folder, create the settings.json file and paste the following:

{
  "eslint.validate": [
    "javascript",
    "javascriptvue",
    "vue"
  ],
  "eslint.format.enable": true,
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  "workbench.colorTheme": "Default Dark+",
  "editor.fontFamily": "Cascadia Code",
  "editor.fontWeight": "400",
}

This will auto-format your code correctly ON EVERY SINGLE SAVE - this is huge. It will fix and format your code according to best practices. This also means you will learn how to write code that's clean, readable, and according to best practices! This is A HUGE trait when working with other people.

The following:

"editor.fontFamily": "Cascadia Code",

Sets the editor font. You can choose other fonts if you wish. To download the above or other suggested fonts, check out the "Fonts" in this document.

My Setup

I prefer to have the "Activity Bar" and "Status Bar" hidden.

I use shortcuts mostly. Don't learn all of them, it's enough to remember those to hide/open the Side Bar, the Bottom Panel (with Terminal), to quickly "Go to File" or the "Command Palette".

I'm not ashamed that I can't remember many shortcuts and mostly use the mouse for everything. Don't worry too much about that too, just use whatever you're comfortable with.
